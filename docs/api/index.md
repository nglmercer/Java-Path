# API Reference

This section details the core modules and functions available in `java-path`.

## Modules

- [**Environment (`env`)**](./env.md): Platform and architecture detection.
- [**Java Information**](./java-info.md): Retrieving Java version info and managing local installations.
- [**Java Service**](./java-service.md): Advanced operations like fetching available versions from Adoptium and downloading releases.
- [**Task Manager**](./task-manager.md): Managing async tasks (download, unpack, backup).
- [**File & Folder Utils**](./files.md): File system helpers.
- [**Command Utils**](./commands.md): Shell command execution and package manager detection.
- [**Validation**](./validation.md): Standardized response types.
